# ML Observability Dashboard

Cloud-native ML observability and monitoring dashboard built using Python, Kubernetes, and distributed infrastructure concepts.

## Overview
This project focuses on monitoring ML infrastructure workflows, tracking service health, and improving operational visibility across distributed systems.

## Features
- ML service monitoring
- Infrastructure observability
- Metrics and logging workflows
- Kubernetes deployment support
- Distributed backend services
- REST API integration

## Tech Stack
- Python
- FastAPI
- Docker
- Kubernetes
- Prometheus
- Grafana
- Linux
